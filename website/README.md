# ShibaPepe Coin Website

This is the official website for the ShibaPepe Coin token on Solana.

## Setup

1. Install Node.js (version 14 or higher)
2. Clone this repository
3. Navigate to the website directory:
   ```bash
   cd website
   ```
4. Install dependencies:
   ```bash
   npm install
   ```

## Running the Website

1. Start the server:
   ```bash
   npm start
   ```
2. Open your browser and navigate to:
   ```
   http://localhost:3001
   ```

## Features

- Connect Solflare wallet
- View token information
- Purchase SHIPEP tokens
- Real-time price calculation

## Security Notes

- This website runs on Solana Devnet for testing purposes
- Always verify transactions before signing
- Never share your private keys
- Make sure you're using the official Solflare wallet

## Development

The website consists of:
- `index.html`: Main website interface
- `styles.css`: Website styling
- `app.js`: Frontend JavaScript
- `server.js`: Backend server
- `package.json`: Project configuration
