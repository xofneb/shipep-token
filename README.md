## 🛠️ Development

### Prerequisites

- Node.js 20.x or higher
- npm or yarn
- Solana CLI (optional)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/xofneb/shipep-token.git
cd shipep-token
```

2. Install dependencies:
```bash
npm install
```

3. Set up environment variables:
```bash
cp env.example .env
# Edit .env with your private key (NEVER commit this file!)
```

### Available Scripts

- `npm start` - Run the main application
- `npm run update-metadata` - Update token metadata
- `npm run check-token` - Check token information
