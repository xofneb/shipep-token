## Overview

ShibaPepe (SHIPEP) is a Solana-based token that combines the beloved Shiba Inu and Pepe meme cultures in the cryptocurrency space. This repository contains the token's metadata, assets, and technical documentation.

## Token Details

- **Name:** ShibaPepe Coin
- **Symbol:** SHIPEP
- **Network:** Solana
- **Token Address:** `HBUgs6vyK3Vj59tD4w2u6QpUL3rTFyJwfe6TeXdkDXhd`
- **Decimals:** 9
